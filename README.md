# GCP Data Engineering Foundations

Repository reorganized by content type for easier navigation.

## Structure

- `notebooks/` — Learning notebooks, setup guides, and execution walkthroughs.
- `scripts/` — Standalone Python scripts.
- `data/` — Local datasets and text files used by exercises.
- `configs/` — YAML/template configuration files.
- `projects/` — Subprojects and module-based examples.
- `docs/` — Documentation and legacy README files.

## Quick Start

1. Open `notebooks/settings_gcp_en.ipynb` or `notebooks/settings_gcp_ptbr.ipynb`.
2. Use Dataproc step notebooks in `notebooks/` for cluster/job flow.
3. Explore module projects under `projects/`.

## Notes

- `.env` is kept at repository root.
- `venv/` is local environment and should stay local-only.
